1.Introducción

La inferencia estadística tiene como objetivo extraer conclusiones sobre una población a partir de la información proporcionada por una muestra. A diferencia de la estadística descriptiva, que se limita a resumir los datos observados, la inferencia incorpora la incertidumbre asociada al muestreo.

Mediante técnicas de estimación y contraste de hipótesis es posible aproximar el valor de parámetros poblacionales desconocidos y evaluar afirmaciones sobre los mismos con un determinado nivel de confianza.

En esta entrada se introducen los conceptos básicos de la inferencia estadística utilizando el lenguaje de R.

2. Inferencia para una población

Para este caso se utilizará el conjunto de datos mtcars, que contiene información sobre distintos modelos de automóviles.

Nos centraremos en la variable mpg, que representa el consumo de combustible.

En el contexto de la inferencia estadística para una población, se considera que se dispone de una muestra aleatoria extraída de una población con un parámetro desconocido. A partir de dicha muestra se utilizan estimadores para aproximar el valor del parámetro poblacional.

En este caso, se analizará la media poblacional del consumo de combustible, utilizando la media muestral como estimador.

data(mtcars)

variable <- mtcars$mpg

n <- length(variable)
n
[1] 32
media_muestral <- mean(variable)
media_muestral
[1] 20.09062
desv_tip <- sd(variable)
desv_tip
[1] 6.026948

La muestra analizada está formada por 32 observaciones. La media muestral del consumo de combustible es de 20.09 , mientras que la desviación típica muestral es de 6.03.

Estos valores resumen la información contenida en la muestra y nos prepara para realizar inferencias sobre la media poblacional del consumo de combustible.

Intervalo de confianza para la media poblacional
IC_media <- t.test(variable, conf.level = 0.95)

El intervalo de confianza al 95 % para la media poblacional del consumo de combustible está comprendido entre 17.92 y 22.26, esto significa que, con un nivel de confianza del 95 %, se puede afirmar que la media poblacional del consumo de combustible se encuentra dentro de dicho intervalo.

Contraste de hipótesis

Para contrastar hipótesis sobre la media poblacional del consumo de combustible, se plantea el siguiente contraste bilateral:

  • Hipótesis nula (H0): la media poblacional del consumo de combustible es igual a 0.
  • Hipótesis alternativa (H1): la media poblacional del consumo de combustible es distinta de 0.

El contraste se realiza utilizando un nivel de significación del 5 %.

contraste <- t.test(variable)
contraste

    One Sample t-test

data:  variable
t = 18.857, df = 31, p-value < 2.2e-16
alternative hypothesis: true mean is not equal to 0
95 percent confidence interval:
 17.91768 22.26357
sample estimates:
mean of x 
 20.09062

El p-valor obtenido es prácticamente nulo y claramente inferior al nivel de significación del 5 %. Por tanto, se rechaza la hipótesis nula y se dice que existe evidencia estadística suficiente para afirmar que la media poblacional del consumo de combustible es distinta de 0.

Comparación de dos poblaciones

En muchas situaciones resulta interesante comparar el comportamiento de una variable entre dos poblaciones distintas. En este caso, se analizará si existen diferencias estadísticamente significativas en el consumo medio de combustible entre vehículos con transmisión automática y transmisión manual.

mpg_auto <- mtcars$mpg[mtcars$am == 0]
mpg_manual <- mtcars$mpg[mtcars$am == 1]

length(mpg_auto)
[1] 19
length(mpg_manual)
[1] 13
mean(mpg_auto)
[1] 17.14737
mean(mpg_manual)
[1] 24.39231

La muestra se divide en 19 vehículos con automáticos y 13 vehículos manuales. La media muestral del consumo de combustible es de 17.15 para los vehículos automáticos y de 24.39 para los vehículos manuales.

A nivel descriptivo, se observa que los vehículos manuales presentan un consumo medio menor que los vehículos automáticos. No obstante, para determinar si esta diferencia es estadísticamente significativa es necesario realizar un contraste de hipótesis.

comparacion <- t.test(mpg_auto, mpg_manual, var.equal = FALSE)

El contraste de medias nos da un p-valor inferior al nivel de significación del 5 %, por lo que se rechaza la hipótesis nula de igualdad de medias.

El intervalo de confianza al 95 % para la diferencia de medias se encuentra entre −11.28 y −3.21, lo que indica que el consumo medio de los vehículos manuales son significativamente menores que el de los vehículos automáticos.

La inferencia estadística es una herramienta fundamental para extraer conclusiones sobre una población a partir de la información muestral, incorporando la incertidumbre al proceso. En esta entrada se han vistolos principales conceptos de la inferencia para una población, incluyendo los intervalos de confianza y los contrastes de hipótesis.
